# 🌶️ Nusantara Bites

**Nusantara Bites** is a simple website project built using **HTML, CSS,
and vanilla JavaScript**. The goal is to bring the vibe of Indonesian
culinary culture into a lightweight, clean, and easy-to-develop web
interface.\
Perfect for learning, experimenting, or as a foundation for simple web
projects.

- 🌴Preview: https://nusantara-bite.vercel.app/

------------------------------------------------------------------------

## ✨ Key Features

-   🎨 **Lightweight and responsive design**\
    Displays well on both desktop and mobile devices.

-   🧭 **Organized folder structure**\
    Neatly separated into `img`, `pages`, `js`, and `styles` folders for
    easier development.

-   ⚡ **Simple JavaScript interactions**\
    Adds basic interactivity without heavy frameworks.

## 🚀 Getting Started

1.  Clone this repository:

    ``` bash
    git clone https://github.com/baskororinakit/Nusantara-Bites.git
    ```

2.  Navigate into the project folder:

    ``` bash
    cd Nusantara-Bites
    ```

3.  Open `index.html` directly in your browser\
    (or use **Live Server** for a smoother workflow).
